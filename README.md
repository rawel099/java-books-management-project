・こちらのアプリの機能について
職業訓練校にて卒業制作として作成した物になります。動作想定としては以下になります。

想定動作
管理する書籍を登録する。（管理者でログインして登録）
登録した書籍はhome画面で一覧として表示され、貸出日時と貸出可能か確認できる。
ログインユーザーを管理し管理者とユーザーで表示内容を管理。（ユーザー追加削除は機能として専用ページへ遷移）
ユーザーでログイン時、貸出は貸出日、貸出ユーザーを登録し返却予定日（２週間後）も登録。
貸出書籍の返却は管理者が本をチェックして問題がなければ返却される形としている。
検索は検索画面へ遷移し、任意の文字列で書籍のタイトルと貸出状態を表示する。
管理者側でメッセージを登録すると、ユーザーがアプリログイン時そのメッセージが表示される。
